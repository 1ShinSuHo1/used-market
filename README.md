# 🧠 Used-Market : AI 기반 중고 전자기기 거래 플랫폼

> AI 품질 분석 + 실시간 경매 + 채팅 기능이 통합된 지능형 중고거래 서비스

---

## 🚀 Overview

**Used-Market**은 중고 전자기기의 **외관 상태를 AI가 자동 등급화(A/B/C)** 하고,  
이를 기반으로 **실시간 경매 및 일반 중고거래**를 지원하는 플랫폼입니다.  
사용자는 AI가 분석한 품질 정보를 신뢰 기반으로 거래할 수 있습니다.

---

## 🔍 주요 기능 (Features)

- 🧠 **AI 품질분석**: FastAPI + PyTorch 기반 MobileNetV2 모델
- 💸 **실시간 경매**: STOMP WebSocket + Redis Pub/Sub 구조
- 💬 **실시간 채팅**: Redis 채널 기반 STOMP 통신
- 📱 **일반 중고거래**: AI 등급 기반의 C2C 거래 기능
- 🧑‍💼 **관리자 기능**: 사용자 신고 및 거래 내역 관리
